# Financias App

Plataforma estática de organização financeira pessoal com dashboard integrado, visão de cartões, calendário inteligente, contas bancárias, despesas e entradas.

## Como visualizar

1. Instale as dependências de um servidor simples (opcional) ou utilize qualquer extensão de "Live Server".
2. No terminal, execute:

```bash
python3 -m http.server 8080
```

3. Acesse `http://localhost:8080` no navegador.

Todo o layout foi construído com HTML, CSS e JavaScript puro para manter uma experiência leve, limpa e responsiva dentro da paleta branca e azul solicitada.
